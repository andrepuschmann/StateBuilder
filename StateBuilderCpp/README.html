The finite state machine source code generator for C++

From a description in XML of a finite state machine, StateBuilderCpp generates the C++ source code.

For a complete list of features, a getting started guide and numerous examples, please have a look at the online documentation or open the offline documentation in CHM format from the startup menu.

Other languages such as C#, VB.NET and Java are also available, have a look a the website
